# claude-scripts

Personal automation scripts, managed and executed by [Claude Code](https://claude.ai/code).

## Scripts

| Script | Description |
|--------|-------------|
| [duke](scripts/duke/) | Checks the next free appointment at Duke of Haircuts (Nürnberg) |

## Usage

```bash
# Install dependencies for a script
cd scripts/<name> && npm install

# Run a script
node scripts/<name>/index.js
```
