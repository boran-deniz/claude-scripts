# duke — Friseur Termin-Checker

Checks the next available appointment at **Duke of Haircuts** in Nürnberg via [Salonkee](https://salonkee.de/salon/duke-of-haircuts).

## Service

- **Category:** MEN
- **Service:** Cut & Finish — Haare ab Schulter (unter Schulter Länge)
- **Duration:** 60 minutes
- **Price:** 90 €

## Usage

```bash
npm install
node index.js
```
